## 🛠️ Installation & Setup

### Prerequisites
- Python 3.10+
- JAX/JAXLib (CUDA/TPU supported)

### Development Install
```bash
git clone https://github.com/deskiziarecords/Synth-fuse.git
cd Synth-fuse
pip install -e .[all]
```

### CLI Usage
```bash
# Start the Cabinet with ingestion watcher
synthfuse

# Check version
synthfuse --version
```
